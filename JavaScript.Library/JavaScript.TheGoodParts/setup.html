1 - JavaScript

It's best practice to separate your concerns and keep your JavaScript code in a separate file. Here's how we do that:

JS file set up

  1. Create an HTML file
  2. Set up your HTML head and body.
  3. Create a js file. For now, save it on the same level.
  4. Add a javascript script tag right above the closing body tag of your html file. It looks like this:
    5. <script type="text/javascript" src="scripts.js"></script>
  5. Add this to your scripts.js file: alert("It works");
  6. To test right click on your index.html window and click open in browser.
  7. You should get an alert message that pops up with your alert message

Most common errors include the the following:

  • You did not save your file(s).
  • You have a typo or spelling error in your script tag.
  • Your JavaScript file is not on the correct level.
