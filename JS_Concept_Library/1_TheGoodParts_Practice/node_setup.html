2 - Node Set up

We'll start out using node to create a javascript console in Sublime.

Set up a Build System

  1. You'll need to install Node.JS
  2. In Sublime -> Go to Tools -> Build System -> New Build System
  3. Windows Users change the default to this:
    {
    "cmd": ["node", "$file"]
    }
  4. Mac Users might need this:
    {
    "cmd": ["/usr/local/bin/node", "$file", "$file_base_name"],
    "working_dir": "${project_path:${folder}}",
    "selector": "*.js"
    }
  5. Comment out your test alert function.
  6. Enter console.log("Hello node!");
  7. Click Ctrl b on Windows or Cmd + b on a Mac
  8. If that doesn't work, go to the Tools menu and click Build.
  9. You should see a console down below.
